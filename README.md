# Open Agent Threat Format (OATF)

A YAML-based specification for describing security threats against AI agents communicating over MCP, A2A, and AG-UI.

**Website & Specification:** [oatf.io](https://oatf.io)

**JSON Schema:** [oatf.io/schemas/v0.1.json](https://oatf.io/schemas/v0.1.json)

## Repository Structure

| Path | Contents |
|---|---|
| `docs/` | Astro Starlight site — canonical source of all specification content |
| `schemas/` | JSON Schema source (`v0.1.json`) |
| `conformance/` | Data-driven YAML test fixtures for SDK conformance |
